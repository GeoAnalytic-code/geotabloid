How to use this site

1. Get the Geopaparazzi App 2. Set up cloud profiles 3. Collect some tracks and images 4. Upload your user project 5. View your content

Get the Geopaparazzi App from the play store

Install Geopaparazzi on your android device.

Download the app from the play store.

Set up cloud profiles

You will need a username and password from this site, sign up here. In the app, navigate to Settings - Cloud Server Settings and enter your username and password. Be sure to set the Cloud Profiles URL as shown.

Now from the main page of the app, select the 'Download' button on the lower left...

choose 'Cloud Profiles'...

It may take a while to download if there is a large map file, but you will get a confirmation when the download is complete.

Collect some tracks and images

Returning to the main menu, the left middle icon starts recording a GPS track...

At any time, you can click the upper left icon to record a note which may include photos, sketches or snapshots of the current map view....

Feel free to play around with the controls, you can add a text description to any note...

Upload your user project

Return to the main menu and select the 'Upload' icon from the lower right, then select 'Cloud Profiles'...

You will see a list of the profiles on your device, select the one that you have been using and the collected data will be uploaded to the server...

It may take a while to upload all the data you have collected. This step is best performed when you have a wifi connection to save on mobile data...

You will see a confirmation when your upload is complete.

View your content

That's it! Now you can view your tracks and notes here on GeoTabloid!
